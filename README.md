# GI_server_switcher

练手用小项目，没有更新的打算

## Usage

将可执行文件放在原神安装目录下
双击运行或命令行启动，首次使用先备份

```
===========================================
|            GI server switcher           |
|              Simple version             |
===========================================
|         Enter the serial num            |
|          1. backup                      |
|          2. restore                     |
|          3. Switch to official server   |
|          4. Switch to bilibili server   |
|          5. exit                        |
===========================================
```

输入3切换至官服，4切换至b服
